The guide for RoLocate

Overview

RoLocate is a free userscript that adds many features but most populat of them all is the server filtering features.

Note: All data processing is done locally in your browser.

Key Benefits

  • Local Execution: Everything is done locally in your browser.
  • Open Source: Source code is available on greasyfork!
  • Regular Updates: Maybe idk lmao don't trust this.

Features

Server Filters

Filter through servers. You can even find the regions of each server!

Show Old Greeting

Shows the old greeting Roblox had on their home page.

Disable Trailer Autoplay

Prevents trailers from autoplaying on Roblox game pages.

Remove All Roblox Ads

Blocks most ads on the Roblox site.

Game Quality Filter

Removes games from the charts/discover page based on your settings.

Quick Navigation

Ability to add quick navigations to the leftside panel of the Roblox page.

Getting Started

After installing the userscript it will start working on Roblox.com

First Launch

The first time you go on to roblox.com with the userscirpt a popup will appear telling you about the recent updates. You can close this.

Quick Start: All settings can be changed on the settings menu on the top right corner.

Interface Overview

Here are where the RoLocate buttons are located:

  • Settings Menu: Top right corner. Looks like the RoLocate logo.
  • Filter Button: The Filter button is available on games under the server tab. It will be right next to "Exclude Full Servers."
  • Recent Servers: Will be located on the server tab on games and under the private server list.
  • Quick Navigation: On the Left side of the roblox page.

Server Filters

RoLocate allows you to filter through servers easily.

Filtering Features:

  • Smallest Servers: Shows emptiest servers.
  • Available Space: Shows servers with space.
  • Player Count: Find servers with a specific player count.
  • Random Shuffle: Randomly show servers.
  • Server Region: Shows a variety of server locations to join.
  • Best Connection: Automatically join the fastest server.
  • Join Small Server: Join a small server.
  • Newest Server: Find servers that are less than 5 minutes old.

Troubleshooting

Common Error codes and issues

Filtering Error: Failed to obtain permission to send API requests to the Roblox API. Please allow the script to enable request sending.
  • 1. Reinstall the userscript manager.
  • 2. Install RoLocate.
  • 3. Go to any roblox game's page.
  • 4. Click on server tab then filter button.
  • 5. Click on server region.
  • 6. Proceed until 2 Tampermonkey/Violentmonkey tabs open up.
  • 7. Click "Always Allow" on both of them.
  • The issue should now be fixed.
Error: No servers found. Please try again later.
  • This usually means that the game is so popular that before RoLocate could scan the servers, the servers fill up to the max player count.
  • Try again later or try a different game.
  • If issue still persists then report on Greasyfork.
Error: Could not estimate location. Fatal error, please report on Greasyfork. Using default (New York).
  • Usually happens if you have a vpn enabled or your on a browser that blocks the request.

Performance Issues

  • Reduce the update interval in settings
  • Disable unused features to improve performance
  • Clear cached data if the script becomes slow
  • Check for browser extensions that might conflict

Debug Mode: Enable debug mode in advanced settings to get detailed console logs for troubleshooting.

Data Not Saving

  • Check browser storage permissions
  • Ensure private/incognito mode isn't interfering
  • Verify sufficient disk space is available
  • Try exporting and re-importing your settings
